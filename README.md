# goit-js-hw-11

A Vite-based image search project that integrates the Pixabay API, renders result cards dynamically, and provides user feedback for loading and empty search states.

## Live Demo

https://vitaliifedunyk.github.io/goit-js-hw-11/

## Overview

This project demonstrates working with asynchronous API requests in a frontend application. It allows users to search for images, fetches results from Pixabay with `axios`, and renders a gallery enhanced with `SimpleLightbox` and notification states via `iziToast`.

## Key Features

- image search powered by the Pixabay API;
- asynchronous requests with `axios`;
- dynamic gallery rendering with `SimpleLightbox`;
- loader and error feedback for empty or failed searches.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Vite
- Axios
- SimpleLightbox
- iziToast

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/vitaliifedunyk/goit-js-hw-11.git
```

2. Open the project folder.
3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```
