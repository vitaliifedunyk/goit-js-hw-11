## vanilla-template

## 📄 Опис

Базовий стартовий шаблон для невеликих фронтенд-проєктів на vanilla HTML, CSS та
JavaScript.

## 🚀 Як використовувати

```bash
git clone https://github.com/vitaliifedunyk/goit-js-hw-11.git
cd vanilla-template
npm install
npm run dev
```

Після запуску відкрийте локальну адресу, яку покаже Vite у терміналі.

## 🛠 Технології

- HTML5
- CSS3
- JavaScript

## 📁 Структура проєкту

```text
.
├── src/
│   ├── index.html
│   ├── main.js
│   ├── css/
│   ├── js/
│   ├── partials/
│   └── public/
├── package.json
└── vite.config.js
```
